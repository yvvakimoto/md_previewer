# Introduction

Welcome to the **sample workspace**. The sidebar on the left shows every
Markdown file in this folder. Click a file to load it.

This file order comes from `_toc.md`. Try renaming it to `_toc.md.bak` and
reload — the previewer will fall back to alphabetical order with directories
first.

## What to try

- Click [Getting Started](chapters/01-getting-started.md) to jump to the next file.
- Press <kbd>Alt</kbd> + <kbd>←</kbd> to go back.
- Press <kbd>E</kbd> to export the whole workspace as a folder of `.html` files.

## Unlisted file

The file `extras.md` is intentionally **not** referenced in `_toc.md`. It still
appears in the tree under the "Other" group at the bottom.
